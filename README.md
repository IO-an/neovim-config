# neovim-config

Минималистичная конфигурация Neovim в духе Vim с современными инструментами.

## Ключевые возможности
- Тёплая тёмная тема Gruvbox.
- Умная подсветка синтаксиса через Tree-sitter (включая styled-components и .env).
- Автодополнение (только LSP и пути, без сниппетов) и диагностика для 16+ языков,
  с ESLint и Stylelint.
- Поиск файлов и текста с превью (Telescope) – с режимом «все файлы».
- Файловый менеджер Oil с показом скрытых файлов.
- Git-индикация ASCII-символами и подсветкой номеров строк.
- Умное комментирование, автоформатирование (Prettier + ESLint/Stylelint).
- Командная строка внизу, без всплывающих окон для `:`.
- Ошибки отображаются как `E` вместо номера строки; подробности – при наведении.
- Счётчик ошибок и предупреждений в статус-строке.
- Лаконичные плавающие окна в стиле VSCode (диагностика, ховер, сигнатура).
- Автоматическая сигнатура функции при вводе аргументов.
- Контекстное меню (правая кнопка мыши или `<leader>m`) с копированием, вставкой,
  быстрыми исправлениями, переходом к определению и пр.
- Поддержка Neovide с единой переменной для анимаций и кроссплатформенным шрифтом.
- Полное отключение временных файлов (swap, backup, undo, shada).
- Ленивая загрузка плагинов — быстрый старт.
- Кроссплатформенная установка одной командой.

## Быстрая установка

### Linux / macOS
```bash
curl -fsSL https://raw.githubusercontent.com/io-an/neovim-config/main/install.sh | bash
```

### Windows (PowerShell)
```powershell
Invoke-Expression (New-Object Net.WebClient).DownloadString('https://raw.githubusercontent.com/io-an/neovim-config/main/install.ps1')
```

## Обновление
Повторный запуск скрипта обновит конфигурацию и плагины.

## Настройки Neovide
Шрифт: Hurmit Nerd Font Mono (Linux/macOS), Consolas (Windows), 14pt.

Анимации: управляются переменной g:neovide_animations_enabled (по умолчанию true).

Прозрачность: 100% (настраивается в lua/config/neovide.lua).

## Поддержка ESLint и Stylelint
ESLint и Stylelint включаются автоматически для JS/TS и CSS/SCSS проектов,
если соответствующие пакеты установлены. Серверы устанавливаются Mason'ом.

## Требования (проверяются скриптом установки)
- Git
- Node.js (v14+)
- Python 3
- ripgrep (rg)
- Компилятор C (cc, gcc или clang) для сборки tree-sitter парсеров
- Для Java, C#, Scala, Pascal — соответствующие SDK устанавливаются отдельно
